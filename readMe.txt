Последовательность действий для запуска ПП
0. Проект скачать: https://github.com/NikitaPishulenak/conference
1. Создать БД в mysql с именем conference
2. Импортировать в нее .sql файл 
3. В connect.php изменить настойки БД (логин. пароль)
4. В selectResult.php s строке 42 изменить путь
5. Перейти в selectResult.php и нажать "Извлечь" предварительно удалив папку ConferenceData
--------------------------------------------------------
index.php-основная форма программы
info.php- страница с уведомлением об успешной регистрации
insertRandomData.php- скрипт для тестового заполнения БД (в итоговом продукте он не нужет)
phpWord.php- сторонняя библиотека для работы с документами
selectResult.php- скрипт по извлечению данных из mysql и импортирование в .svc и .doc
sendMail.php- скрипт для отпраки сообщения с уведомлением на почту зарегистрировавшегося
template.docx- шаблон по которому генерируются дакументы
********************************************************

Все извлеченные данные помещаются в папку ConferenceData (..\conference\PHP\ConferenceData)
а дальше создаются папки с названиями секций конференции в которых и расположены соответствующие документы (.csv и .doc)
